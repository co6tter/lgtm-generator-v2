## 🎯 プロジェクト概要

### プロジェクト名
**LGTM Generator v2**

### 目的
開発者が Pull Request のレビュー時に使用する LGTM (Looks Good To Me) 画像を簡単に生成・共有できる Web アプリケーションを提供する。

### 主要機能
1. **画像検索**: Unsplash / Pexels / Pixabay から画像検索
2. **LGTM生成**: 選択した画像に "LGTM" テキストを重ねる
3. **マークダウンコピー**: GitHub/GitLab で使える形式でコピー
4. **画像ダウンロード**: ローカルに保存

### 技術スタック
- **Frontend**: Next.js 14+ (App Router), React 18+, TypeScript, Tailwind CSS
- **Backend**: Next.js API Routes (Serverless)
- **Hosting**: Vercel (無料プラン)
- **External APIs**: Unsplash, Pexels, Pixabay
